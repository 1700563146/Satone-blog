这个作品用于探索“个人网站能否像一张电影海报一样打开”。页面结构尽量克制，让背景承担情绪，文字与按钮只作为漂浮在画面上的轻量引导。

## 设计思路

首屏使用大面积视觉背景，信息放在屏幕中下部，让顶部留白成为画面的一部分。玻璃面板负责保证文字可读，同时不破坏背景的沉浸感。

## 技术实现

页面基于 Astro 静态生成，内容由 Markdown 管理。视觉层主要依靠 CSS 变量、渐变、模糊、阴影和轻量交互动效完成。
